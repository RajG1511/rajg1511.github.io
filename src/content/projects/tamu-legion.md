## What it is

A SaaS portal that centralizes operations for a Texas A&M student organization, with
strict role-based access control. I led the Agile development as project manager.

## How it works

- **Ruby on Rails** on **PostgreSQL**, containerized with **Docker**.
- **GitHub Actions CI/CD** pipelines gate every merge on linting and unit tests with
  **90%+ code coverage** — the pipeline is the reviewer that never gets tired.
